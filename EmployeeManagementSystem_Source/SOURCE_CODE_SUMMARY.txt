-------------------------

BUILD INSTRUCTIONS:
-------------------
1. Open ManagementApp.sln in Visual Studio
2. Build the solution (Ctrl+Shift+B)
3. Run applications from Visual Studio or command line

Command Line:
- dotnet build ManagementApp.sln
- dotnet run --project ManagementApp/ManagementApp.csproj
- dotnet run --project DisplayApp/DisplayApp.csproj
